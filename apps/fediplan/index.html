FediPlan

Web (self-hosted)
Web PHP GPL v3
language Website code Source code

FediPlan is a web application to schedule posts on Mastodon and Pleroma. Log in with your Fediverse account, compose your message, pick a date and time, and FediPlan publishes it for you. You can use it on the official instance or host your own.

A mobile version is also available: FediPlan App.

Key features

  • Schedule posts: pick a date and time, FediPlan handles the rest
  • Multi-account: manage and schedule across multiple Fediverse accounts
  • Post editor: compose with media attachments, polls, content warnings, and visibility settings
  • Manage scheduled posts: view, edit, and cancel upcoming publications
  • Mastodon and Pleroma: compatible with both platforms via OAuth
  • Self-hostable: deploy your own instance with Docker or a manual PHP setup
  • Translations: community-translated via Crowdin in 25+ languages
