Fedilab lets you access multiple Fediverse accounts from a single Android app. Whether you use Mastodon, Pleroma, Pixelfed, PeerTube, or Misskey, you can manage all your accounts in one place without switching between apps.

## Supported platforms

| Platform | Support |
|----------|---------|
| Mastodon | Full |
| Pleroma / Akkoma | Full, with reactions |
| Misskey | Full |
| PeerTube | Dedicated video module |
| Pixelfed | Via Mastodon API |
| Lemmy | Public timelines |
| GNU Social | Public timelines |

## Key features

- **Multi-account**: manage accounts across different instances and platforms
- **Timelines**: Home, Local, Public, Hashtag, and Trending
- **Post editor**: compose with media, polls, content warnings, and scheduling
- **Conversations**: direct messages with threaded view
- **Cross-posting**: share content across your accounts in one action
- **Thread tracking**: highlights new replies since your last visit
- **Built-in browser**: open links without leaving the app
- **Translation**: translate posts directly in the timeline
- **Content filters**: fine-grained keyword filtering per context
- **Scheduled posts**: write now, publish later
- **Themes**: Light, Dark, Black, Solarized, Dracula, and dynamic colors on Android 12+
- **Accessibility**: mandatory alt text option for media uploads
- **Privacy-friendly notifications**: UnifiedPush support on F-Droid
- **Proxy support**: configure custom network settings
